# Dossier pour les images de la documentation

Placez ici toutes les images utilisées dans la documentation.

## Structure recommandée

- `ecampusfrance-accueil.jpg` - Capture d'écran de la page d'accueil
- `formulaire-inscription-ecampus.jpg` - Capture d'écran du formulaire
- `recherche-formations-ecampus.jpg` - Capture d'écran de la recherche
- `exemple-diplome-bac.jpg` - Exemple de diplôme (données masquées)
- `exemple-releve-notes.jpg` - Exemple de relevé (données masquées)
- `exemple-photo-identite.jpg` - Exemple de photo d'identité
- `soumission-dossier-ecampus.jpg` - Capture d'écran de soumission
- `entretien-campus-france.jpg` - Photo/illustration d'entretien
- `lettre-admission-exemple.jpg` - Exemple de lettre d'admission
- `consulat-visa-processus.jpg` - Photo du consulat/processus visa
- `residence-etudiante-crous.jpg` - Photo de résidence CROUS
- `colocation-etudiante.jpg` - Photo de colocation
- `studio-etudiant.jpg` - Photo de studio étudiant

## Format des images

- **Format** : JPG ou PNG
- **Taille** : Optimisée pour le web (max 1-2 Mo)
- **Résolution** : 1200-1920px de largeur recommandée
- **Nommage** : Utilisez les noms suggérés ci-dessus

## Confidentialité

⚠️ **Important** : Pour les exemples de documents (diplômes, relevés, lettres), masquez TOUTES les informations personnelles avant de les ajouter.
