La soumission de votre dossier est l'  finale de votre candidature. C'est le moment de vérifier que tout est complet et correct avant de l'envoyer définitivement.

## Vérification finale avant soumission

### Checklist complète du dossier

Avant de soumettre, vérifiez que vous avez :

#### Documents académiques
- [ ] Tous les diplômes (originaux + traductions)
- [ ] Tous les relevés de notes (toutes les années)
- [ ] Certificats de langue (si requis)
- [ ] Documents numérisés en bonne qualité

#### Documents administratifs
- [ ] Passeport valide (copie complète)
- [ ] Acte de naissance (original + traduction si nécessaire)
- [ ] Justificatifs de ressources financières
- [ ] Photos d'identité

#### Documents de candidature
- [ ] Projet motivé rédigé et relu
- [ ] CV (si demandé)
- [ ] Lettres de recommandation (si demandées)
- [ ] Autres documents spécifiques demandés

#### Informations personnelles
- [ ] Profil complété sur la plateforme
- [ ] Informations personnelles à jour
- [ ] Coordonnées correctes (email, téléphone, adresse)

#### Formations sélectionnées
- [ ] Vœux sélectionnés (jusqu'à 15)
- [ ] Ordre de préférence défini
- [ ] Dates limites vérifiées pour chaque formation

### Vérification de chaque document

Pour chaque document, vérifiez :

-  **Qualité** : Le document est lisible et net
-  **Complétude** : Toutes les pages sont présentes
-  **Traduction** : Documents traduits si nécessaire
-  **Légalisation** : Documents légalisés si requis
-  **Format** : Format correct (PDF, JPG)
-  **Taille** : Taille du fichier acceptable
-  **Nommage** : Nom du fichier clair et cohérent

### Vérification des informations

Vérifiez que toutes les informations sont :

-  **Exactes** : Pas d'erreur de frappe
-  **Cohérentes** : Les informations correspondent entre les documents
-  **À jour** : Informations récentes et valides
-  **Complètes** : Aucune information manquante

## Processus de soumission

### 1. Accéder à votre espace candidat

1. Connectez-vous à votre compte E-Campus France
2. Accédez à **"Mes candidatures"** ou **"Mes dossiers"**
3. Vérifiez l'état de chaque candidature

### 2. Vérifier chaque vœu

Pour chaque formation sélectionnée :

1. **Ouvrez** le dossier de candidature
2. **Vérifiez** que tous les documents sont uploadés
3. **Vérifiez** que toutes les sections sont complétées
4. **Vérifiez** les dates limites spécifiques

### 3. Finaliser le dossier

1. **Complétez** toutes les sections obligatoires
2. **Uploadez** tous les documents requis
3. **Vérifiez** une dernière fois chaque élément
4. **Sauvegardez** les modifications

### 4. Soumettre le dossier

1. **Vérifiez** que tout est complet
2. **Lisez** attentivement le message de confirmation
3. **Cliquez** sur **"Soumettre"** ou **"Finaliser"**
4. **Confirmez** la soumission

>  **Attention** : Une fois soumis, certaines modifications ne sont plus possibles. Vérifiez bien avant de confirmer !

## Confirmation de soumission

### Message de confirmation

Après la soumission, vous devriez recevoir :

-  **Message à l'écran** : Confirmation de soumission
-  **Email de confirmation** : À l'adresse email renseignée
-  **Numéro de dossier** : Notez-le précieusement

### Conserver les preuves

-  **Sauvegardez** l'email de confirmation
-  **Notez** le numéro de dossier
-  **Faites une capture d'écran** de la confirmation
-  **Sauvegardez** une copie de votre dossier complet

## Modifications après soumission

### Ce qui peut encore être modifié

Après soumission, vous pouvez généralement encore :

-  **Ajouter des documents complémentaires** (si demandés)
-  **Modifier certaines informations** (selon les établissements)
-  **Contacter l'établissement** pour des questions

### Ce qui ne peut plus être modifié

Une fois soumis, vous ne pouvez généralement plus :

-  **Retirer un vœu**
-  **Modifier l'ordre de préférence** (dans certains cas)
-  **Modifier les documents principaux** (diplômes, relevés)
-  **Changer de formation** pour un vœu

### En cas d'erreur

Si vous avez fait une erreur après soumission :

1. **Contactez immédiatement** l'établissement concerné
2. **Expliquez** la situation
3. **Demandez** si une modification est possible
4. **Fournissez** les documents corrects si accepté

>  **Important** : Les établissements ne sont pas obligés d'accepter les modifications. Agissez rapidement si vous avez fait une erreur.

## Suivi de votre dossier

### Accéder au suivi

1. Connectez-vous à votre compte E-Campus France
2. Accédez à **"Mes candidatures"**
3. Consultez l'**état** de chaque candidature

### États possibles

Votre dossier peut être dans différents états :

-  **En cours** : Dossier en cours de traitement
-  **Complet** : Tous les documents sont reçus
-  **En évaluation** : Dossier en cours d'évaluation
-  **Demande de complément** : Des documents supplémentaires sont demandés
-  **Admis** : Candidature acceptée
-  **Refusé** : Candidature refusée
-  **En attente** : En liste d'attente

### Notifications

Vous recevrez des notifications par email pour :

-  Confirmation de réception du dossier
-  Demande de documents complémentaires
-  Changement d'état du dossier
-  Réponse d'admission ou de refus

### Vérifier régulièrement

-  **Connectez-vous régulièrement** (au moins une fois par semaine)
-  **Vérifiez vos emails** (y compris les spams)
-  **Activez les notifications** si disponibles

## Dates limites importantes

### Dates à retenir

-  **Date limite de soumission** : Date butoir pour soumettre votre dossier
-  **Date limite de documents** : Date limite pour certains documents spécifiques
-  **Date de réponse** : Date prévue de réponse des établissements

### Respecter les délais

-  **Ne tardez pas** : Soumettez votre dossier bien avant la date limite
-  **Vérifiez les dates** : Chaque établissement peut avoir des dates différentes
-  **Prévoyez un délai** : Les problèmes techniques peuvent survenir

### En cas de retard

Si vous êtes en retard :

1. **Contactez immédiatement** l'établissement
2. **Expliquez** la situation
3. **Demandez** une extension (non garantie)
4. **Soumettez** dès que possible

## Erreurs fréquentes et solutions

### Erreur : Dossier incomplet

**Problème** : Certains documents manquent.

**Solution** :
- Vérifiez la liste des documents requis
- Uploadez les documents manquants
- Contactez l'établissement si vous avez un problème

### Erreur : Document rejeté

**Problème** : Un document n'est pas accepté.

**Solutions** :
- Vérifiez le format et la qualité
- Vérifiez que le document est complet
- Contactez l'établissement pour comprendre le problème
- Uploadez une nouvelle version corrigée

### Erreur : Soumission après la date limite

**Problème** : Vous avez soumis après la date limite.

**Solution** :
- Contactez immédiatement l'établissement
- Expliquez la situation
- Demandez si une exception est possible (non garanti)

### Erreur : Erreur dans les informations

**Problème** : Vous avez fait une erreur dans les informations.

**Solution** :
- Contactez l'établissement rapidement
- Expliquez l'erreur
- Demandez si une correction est possible
- Fournissez les informations correctes

## Checklist de l'  6

Avant de considérer votre dossier comme soumis, assurez-vous d'avoir :

- [ ] Tous les documents uploadés et vérifiés
- [ ] Toutes les sections complétées
- [ ] Informations personnelles vérifiées et correctes
- [ ] Vœux sélectionnés et ordre de préférence défini
- [ ] Dates limites vérifiées pour chaque formation
- [ ] Dossier relu une dernière fois
- [ ] Dossier soumis avant la date limite
- [ ] Email de confirmation reçu
- [ ] Numéro de dossier noté
- [ ] Copies de sauvegarde conservées
- [ ] Suivi du dossier activé

## Après la soumission

### Attendre les réponses

-  Les établissements ont besoin de temps pour évaluer les dossiers
-  Les réponses arrivent généralement entre avril et juillet
-  Restez patient et positif

### Préparer l'entretien

Si un entretien est requis :

-  Préparez-vous dès maintenant
-  Consultez l'[  7 : Entretien Campus France](./etape-7-entretien)
-  Rassemblez les documents nécessaires

### Continuer à vous informer

-  Restez informé sur les établissements
-  Préparez-vous pour les prochaines  s
-  Consultez les ressources disponibles
