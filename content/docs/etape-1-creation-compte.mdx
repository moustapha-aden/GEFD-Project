La première   de votre parcours consiste à créer votre compte sur la plateforme E-Campus France. Cette   est fondamentale car elle vous donne accès à tous les outils nécessaires pour votre candidature.

## Accès à la plateforme

### URL de la plateforme

Rendez-vous sur le site officiel : **[www.ecampusfrance.org](https://www.ecampusfrance.org)**

>  **Attention** : Assurez-vous d'être sur le site officiel. Méfiez-vous des sites frauduleux qui pourraient tenter de récupérer vos informations personnelles.

### Navigateurs compatibles

La plateforme fonctionne avec tous les navigateurs modernes :
-  Google Chrome (recommandé)
-  Mozilla Firefox
-  Microsoft Edge
-  Safari
-  Opera

## Processus d'inscription

### 1. Accéder à la page d'inscription

1. Sur la page d'accueil, cliquez sur **"Créer un compte"** ou **"S'inscrire"**
2. Vous serez redirigé vers le formulaire d'inscription

### 2. Remplir le formulaire

Le formulaire demande les informations suivantes :

#### Informations personnelles

- **Nom de famille** : Votre nom tel qu'il apparaît sur votre passeport
- **Prénom(s)** : Votre prénom tel qu'il apparaît sur votre passeport
- **Date de naissance** : Format JJ/MM/AAAA
- **Nationalité** : Votre nationalité actuelle
- **Pays de résidence** : Le pays où vous résidez actuellement

#### Informations de contact

- **Adresse email** :  **Très important** - Utilisez une adresse email valide et à laquelle vous avez accès régulièrement
- **Numéro de téléphone** : Avec l'indicatif pays (ex: +253 pour Djibouti)
- **Adresse postale** : Votre adresse complète

#### Informations de connexion

- **Mot de passe** : 
  - Minimum 8 caractères
  - Contenir des majuscules et minuscules
  - Contenir au moins un chiffre
  - Contenir au moins un caractère spécial
- **Confirmation du mot de passe** : Saisissez à nouveau le même mot de passe

>  **Conseil** : Choisissez un mot de passe fort et unique. Notez-le dans un endroit sûr. Vous en aurez besoin pour vous connecter régulièrement.

### 3. Accepter les conditions

-  Cochez la case pour accepter les conditions d'utilisation
-  Cochez la case pour accepter la politique de confidentialité
- Lisez attentivement ces documents avant d'accepter

### 4. Valider l'inscription

- Cliquez sur **"Créer mon compte"** ou **"Valider"**
- Un message de confirmation devrait apparaître

## Vérification de l'email

### Email de confirmation

Après avoir créé votre compte, vous recevrez un email de confirmation à l'adresse que vous avez renseignée.

###  s de vérification

1. **Vérifiez votre boîte de réception** (et les spams si nécessaire)
2. **Ouvrez l'email** de Campus France
3. **Cliquez sur le lien de confirmation** dans l'email
4. Vous serez redirigé vers la plateforme avec votre compte activé

>  **Délai** : L'email peut prendre quelques minutes à arriver. Si vous ne le recevez pas dans les 15 minutes, vérifiez vos spams ou réessayez.

### Si vous ne recevez pas l'email

1. **Vérifiez vos spams/courrier indésirable**
2. **Vérifiez que l'adresse email est correcte**
3. **Attendez quelques minutes** (les emails peuvent être retardés)
4. **Demandez un nouvel email de confirmation** depuis la page de connexion
5. **Contactez le support** si le problème persiste

## Première connexion

### Se connecter

1. Rendez-vous sur [www.ecampusfrance.org](https://www.ecampusfrance.org)
2. Cliquez sur **"Se connecter"** ou **"Connexion"**
3. Entrez votre **adresse email** et votre **mot de passe**
4. Cliquez sur **"Connexion"**

### Profil à compléter

Lors de votre première connexion, vous devrez compléter votre profil :

- Informations personnelles supplémentaires
- Niveau d'études actuel
- Objectifs de formation
- Niveau de français

>  **Important** : Prenez le temps de remplir correctement toutes les informations. Elles seront utilisées pour votre candidature.

## Erreurs fréquentes et solutions

### Erreur : "Email déjà utilisé"

**Problème** : L'adresse email est déjà associée à un compte.

**Solutions** :
- Si c'est votre compte, essayez de réinitialiser votre mot de passe
- Si vous avez oublié votre mot de passe, utilisez la fonction "Mot de passe oublié"
- Si ce n'est pas votre compte, utilisez une autre adresse email

### Erreur : "Mot de passe trop faible"

**Problème** : Le mot de passe ne respecte pas les critères de sécurité.

**Solution** : Assurez-vous que votre mot de passe contient :
- Au moins 8 caractères
- Des majuscules ET des minuscules
- Au moins un chiffre
- Au moins un caractère spécial (!, @, #, $, etc.)

### Erreur : "Email invalide"

**Problème** : Le format de l'email n'est pas correct.

**Solution** : Vérifiez que votre email contient :
- Un nom d'utilisateur (avant le @)
- Le symbole @
- Un nom de domaine (après le @)
- Une extension (.com, .fr, etc.)

Exemple valide : `prenom.nom@email.com`

### Erreur : "Compte non activé"

**Problème** : Vous n'avez pas cliqué sur le lien de confirmation dans l'email.

**Solution** :
1. Vérifiez votre boîte email
2. Cliquez sur le lien de confirmation
3. Si l'email a expiré, demandez un nouvel email de confirmation

### Problème : Impossible de se connecter

**Vérifications à faire** :
1.  L'adresse email est correcte (pas de faute de frappe)
2.  Le mot de passe est correct (vérifiez les majuscules/minuscules)
3.  Le compte est bien activé (email de confirmation cliqué)
4.  Le compte n'est pas bloqué (trop de tentatives échouées)

**Solution** : Utilisez la fonction "Mot de passe oublié" pour réinitialiser votre mot de passe.

## Sécurité de votre compte

### Bonnes pratiques

-  **Ne partagez jamais** votre mot de passe avec qui que ce soit
-  **Utilisez un mot de passe unique** pour E-Campus France
-  **Déconnectez-vous** après chaque utilisation sur un ordinateur partagé
-  **Vérifiez régulièrement** l'activité de votre compte

### En cas de problème de sécurité

Si vous pensez que votre compte a été compromis :

1. Changez immédiatement votre mot de passe
2. Contactez le support Campus France
3. Vérifiez les informations de votre profil
